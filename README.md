# Bookmark Manager

## User Stories
```
As a user,
So that I can see my bookmarks,
I'd like to be able to see a list of bookmarks.
```

```
As a user,
So that I can include more bookmakrs,
I'd like to be able to add to my list of bookmarks.
```
